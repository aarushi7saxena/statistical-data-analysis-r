# statistical-data-analysis-r
Statistical data analysis in R using tidyverse to clean, transform, and summarize population datasets with grouped descriptive statistics and reproducible reporting in RMarkdown.
Statistical Data Analysis in R
Overview

This project performs quantitative analysis on biological population datasets using R and the tidyverse ecosystem. The goal of the analysis is to apply statistical programming techniques to clean, transform, and summarize data across populations and demographic groupings.

Using grouped analysis and descriptive statistics, the project examines patterns across populations and sex categories while demonstrating reproducible analytical workflows using RMarkdown.

This project highlights core skills relevant to mathematics, actuarial science, and quantitative analysis, including data manipulation, statistical computation, and reproducible reporting.

Methods

The analysis applies several statistical programming techniques:

Data cleaning and preprocessing

Variable transformation using conditional logic

Grouped statistical summaries across populations

Calculation of descriptive statistics (means across numeric variables)

Distributional comparisons between population groups

The workflow follows a reproducible pipeline using RMarkdown to integrate code, results, and documentation.
