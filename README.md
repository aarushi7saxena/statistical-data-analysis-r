# Statistical Data Analysis in R

**Skills Demonstrated:** R • Statistical Analysis • Data Manipulation • Tidyverse • Reproducible Research • Quantitative Modeling

## Overview
This project performs quantitative analysis on biological population datasets using **R** and the **tidyverse** ecosystem. The analysis applies statistical programming techniques to clean, transform, and summarize structured data across populations and demographic groupings.

Using grouped analysis and descriptive statistics, the project evaluates patterns across populations and sex categories while demonstrating a reproducible analytical workflow using **RMarkdown**. The project highlights skills relevant to **mathematics, actuarial science, and quantitative analysis**, including statistical computation, data manipulation, and reproducible reporting.

---

## Methods
The analysis applies several statistical programming techniques:

- Data cleaning and preprocessing
- Variable transformation using conditional logic
- Grouped statistical summaries across populations
- Calculation of descriptive statistics (means across numeric variables)
- Distributional comparisons between population groups

The workflow follows a reproducible pipeline using **RMarkdown** to integrate code, results, and documentation.

---

## Sample Output
The analysis produces grouped summary statistics across populations and sexes, allowing comparison of mean values for numeric traits within each demographic group. This provides a structured overview of variation across populations and highlights potential statistical differences between regions.

The results are presented in a reproducible analytical report generated through **RMarkdown**, which integrates code, statistical summaries, and interpretation in a single document.
